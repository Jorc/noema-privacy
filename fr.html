Politique de confidentialité de Noema

Dernière mise à jour : 2026-07-01

English简体中文日本語FrançaisEspañolالعربية

Aperçu

Noema est une base de connaissances IA privée sur l'appareil. Elle vous aide à importer du texte, des pages web et des PDF, à les organiser localement et à poser des questions avec des citations de sources.

Données stockées sur votre appareil

L'app stocke localement le contenu importé, le texte extrait, les résumés, les tags, les points clés, les index de recherche, les métadonnées de récupération, les questions, les réponses, les citations, l'historique récent, les réglages locaux et l'état de traitement.

Ces données sont conservées dans le sandbox de l'app ou dans le conteneur App Group afin que l'app principale et l'extension de partage puissent fonctionner ensemble.

Accès réseau

L'app n'utilise pas de compte, de synchronisation cloud, de réseau publicitaire ni de service d'analyse. Elle accède au réseau uniquement lorsque vous choisissez d'importer une URL web.

Collecte de données

La version actuelle ne collecte pas vos documents, questions, réponses, URL, identifiants, données d'utilisation ou diagnostics pour le développeur.

Suivi

L'app ne vous suit pas entre des apps ou sites web appartenant à d'autres sociétés.

Traitement IA local

L'app est conçue pour traiter le contenu importé localement lorsque cela est pris en charge. Si le modèle local n'est pas disponible, elle utilise un repli extractif local au lieu d'envoyer votre contenu à un serveur du développeur.

Suppression des données

Vous pouvez supprimer les données locales dans les réglages de l'app. La suppression de l'app retire aussi son conteneur local selon le comportement d'iOS.

Modifications

Si cette politique change, la nouvelle version sera publiée avec une nouvelle date.

Contact

Pour toute question de confidentialité ou demande d'assistance, contactez le développeur via GitHub Issues.
